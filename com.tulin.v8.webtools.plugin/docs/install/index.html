Installing

Software Requirements

  • Eclipse 3.3.x
    http://www.eclipse.org/
  • Graphical Editing Framework (GEF). This is an optional. If GEF is not installed, palette view is not available.
    http://www.eclipse.org/gef/

Installing the plugon

  1. Download and install EclipseHTMLEditor from https://sourceforge.jp/projects/amateras/files/?release_id=29403#29403.
  2. Unzip the downloaded file.
  3. Put unzipped folder com.tulin.v8.webtools_x.x.x into ECLIPSE_HOME/plugins.
  4. Restart Eclipse.

Note: SWT Browser widget (HTMLEditor uses it for preview HTML and JSP) requires Mozilla in the Linux. See details at The SWT FAQ. Also you can disable preview in the preference dialog. Select Window/Preferences/Amateras from menubar and check Disable Preview. Then you would be able to use HTMLEditor without Mozilla.
